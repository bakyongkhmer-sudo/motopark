# MotoSheet Vtrust Portal UI

This is a separate Vtrust Tower UI version based on the design handoff.

It does not replace the current `web_portal/` app.

## Run

Open `index.html` directly, or deploy this folder as a separate Vercel project/root directory.

## Notes

- Uses the existing Supabase project for registered motors and daily reports where available.
- Falls back to Vtrust seed-style data for dashboard, companies, and settings previews.
